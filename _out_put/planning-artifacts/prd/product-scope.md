# Product Scope

This section provides a high-level overview. See **Project Scoping & Phased Development** for detailed feature tables, phase gate conditions, and risk mitigation strategies.

- **MVP (14 weeks):** Authentication, FSRS review engine, bilingual EN-JP vocabulary cards, structured vocabulary request, auto-enrichment with JMdict validation, personal collections, CSV import, basic learning diagnostics, progress dashboard, pre-seeded corpus (3,000–5,000 terms), freemium tier gating.
- **Growth (Weeks 15–18):** Chat-based vocabulary request, browser extension, Anki APKG/Quizlet import, enhanced diagnostics, shareable reports.
- **Vision (6+ months):** Four-skill expansion, Korean-English corridor, knowledge graph, B2B/Corporate, marketplace, mobile native apps, multi-signal SRS adaptation.
